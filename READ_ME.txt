you can test itoa, ft_atoi, ft_strlen, ft_is_alpha, ft_lstsize functions with keywords : "strlen" "lstsize" "itoa" "atoi" "isalpha"
you can test functions of your choice ; if no function is declared, all of them will be tested.
you can choice the tests with keywords : "basic" "NULL" "big"
basic: basic inputs are tested.
NULL: NULL input is tested.
big: tested with MAX_INT or MIN_INT or big line for exemple.
if no test is declared : all test will be executed.
the choice of tests will be applicated to all of functions you want to test.
